# Trove AI - Web Development

A professional freelance website showcasing web development services and portfolio.

## Features

- **Modern Design**: Clean, professional layout with gradient accents
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth animations, hover effects, and transitions
- **Service Showcase**: Detailed presentation of freelance web development services
- **Portfolio Gallery**: Project examples with technology stacks
- **Contact Form**: Functional contact form with validation
- **Navigation**: Smooth scrolling navigation with mobile hamburger menu

## Technologies Used

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Modern styling with flexbox and grid layouts
- **JavaScript**: Interactive features and form handling
- **Font Awesome**: Professional icons
- **Google Fonts**: Inter font for modern typography

## Website Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **Services**: Six key freelance web development services
3. **Portfolio**: Recent project showcase with tech stacks
4. **About**: Professional bio and skills overview
5. **Contact**: Contact information and functional contact form

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website is ready to view locally

## Customization

### Updating Personal Information
- Update name, email, and phone in the contact section
- Modify the about section content
- Add your own portfolio projects

### Styling
- Main color scheme uses purple gradients (#667eea to #764ba2)
- Primary accent color: #6366f1
- Font: Inter (Google Fonts)

### Contact Form
The contact form currently simulates submission. To make it functional:
1. Replace the form submission logic in `script.js`
2. Add backend endpoint for form processing
3. Configure email sending functionality
